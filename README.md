## Como executar:

Se voce quer somente vizualizar o site clique [AQUI](https://projeto-scripts.vercel.app/)

Caso queira executa-lo em sua maquina:

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
```

Abra [http://localhost:3000](http://localhost:3000) em seu navegador para visualizar o resultado.
